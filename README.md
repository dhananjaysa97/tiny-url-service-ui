# TinyUrl UI 

# Technologies
* Framework: React 18
* Build Tool: Vite
* State Management: Hooks (and URL Service Managers)
* Testing:** Vitest

# Getting Started

# Prerequisites
* Node.js (v18+)
* npm or pnpm

# Installation
1. Navigate to this directory:
   ```bash
   cd tiny-url-service-ui
   npm install

2. Run the app 
   npm run dev 

   # app will run at http://localhost:5173
